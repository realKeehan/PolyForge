Good to know

If you get a warning that the app cannot be opened, click "Run anyways". You may need to click "More info" to see this option. This warning is entirely harmless and only shows because the app is not signed. Signing it would cost upwards of 359€/year.

macOS Gatekeeper

macOS may block unsigned apps. Right-click the app and select "Open" to bypass Gatekeeper on first launch. Alternatively, run xattr -cr /path/to/PolyForge.app in Terminal.

Linux permissions

After downloading an AppImage or ELF binary, you may need to mark it as executable: chmod +x PolyForge-*. Some desktop environments handle this automatically.

Requirements

What you need to run PolyForge.

Operating System

Windows 10 or later (64-bit). Linux and macOS support is planned.

Storage

Minimal disk space - PolyForge itself is lightweight. Modpacks vary in size.

Internet

An active internet connection for downloading modpack files during installation.

Is it safe?

PolyForge is scanned across multiple security analysis platforms including VirusTotal, Hybrid Analysis, ANY.RUN, and more. Some antivirus software may flag new or unsigned executables - this is a common false positive for Go-compiled binaries.
