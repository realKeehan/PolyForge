Coming soon

High-impact features that push PolyForge past "just an installer."

Custom modpack builder

Assemble packs from supported sources, configure mod lists, and export or share them with your community. A visual workflow for pack creation and distribution.

Image placeholder

Delta updates

Only download what changed between pack versions. Efficient binary diffing means faster updates, less bandwidth, and happier users on slow connections.

Image placeholder

Profile backup & restore

One-click backups of configs, saves, and profiles before any update. Restore instantly if something breaks - never lose your settings again.

Image placeholder

Integrity verification

Every file in a pack is hash-verified against the manifest. Detect corruption, tampering, or incomplete downloads before they cause issues in-game.

Image placeholder

Self-updater

PolyForge will update itself with safe rollback support. Binary updates are separate from content - new packs appear without reinstalling the app.

Image placeholder

Password-protected packs

Private packs for teams, events, or communities with password-controlled access. Distribute exclusive content without exposing it to the public.

Image placeholder

Cross-platform support

Full Linux and macOS support is on the roadmap. Path heuristics, launcher discovery, and platform-specific packaging for every major desktop OS.
